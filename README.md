## Pasos 

- Crear proyecto de npm con el comando `npm init`, esto creará el archivo package.json

## Instalación

```
npm install paquete
```

## Uso

```
import { add } from 'paquete'

add(5,3) //8
add(6,2) //8
add(10, 4) //14
```
